# Jeu-du-pendu
Le jeu du pendu est un jeu de réflexion et de vocabulaire où un joueur doit deviner un mot secret, lettre par lettre, dans un nombre limité d’essais et un temps imparti.
Dans cette version, le joueur peut choisir de jouer seul (avec différents niveaux de difficulté) ou à deux joueurs (l’un choisit la catégorie et le mot, l’autre doit deviner).
À chaque mauvaise réponse, une partie du pendu apparaît. Le joueur gagne s’il trouve toutes les lettres avant d’avoir épuisé ses essais ou le temps.
Un système de score prend en compte la rapidité, la longueur du mot et le nombre d’erreurs. Les meilleurs scores sont enregistrés et affichés dans un tableau des scores, permettant à chacun de se mesurer aux autres joueurs.
